 #begginerChallenge コード勉強中です。普段使いしたい各アプリをLeo AIに作ってもらい、意味なども少しずつ教わりながら一カ所に集めています。
 - 背景に流れ星を配置できるよう、htmlに背景色#000（黒）を、bodyは透明にしました。html,bodyを一括で黒色にすると、星の背景が両者の間に挟まれるようで見えなくなったためです。
 - 背景の流星群は、ease-in-outで緩急をつけ、点滅もします。
 - 検索＆置換アプリは、スマホ用に欲しくてLeo AIに作ってもらいました。目に優しいよう色味を変えたり、スペースを取らなくていい「検索する言葉」や「置換する言葉」は一列横並びに詰め込みました。使い勝手がいいよう、対象テキストを一番上に持ってきました。スマホのキーボード出現時も全画面が見えるよう、フレックスボックスにしました。
 - スワップゲームでは、フィッシャー・イェーツ法にて完全ランダムに、矢印でつなげた絵文字とボタンの柄、紐づく関数を変えられます。
 - 単色画像生成アプリ。これもLeo AIに作ってもらったものです。色味は少し変えました。
 - PWA（Progressive Web App）化しました。
 - オフラインでも使えるよう、Service Workerを導入しています。
 - コード倉庫は、Leo AIに教わったアプリを組み込みました。全選択ボタンは省き、色も変更。
