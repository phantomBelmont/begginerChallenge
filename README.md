# begginerChallenge Leo AIから教わり習得したコードを少しずつ追加します。下記は、Leo AIにコードを見せて作ってもらったアプリ内容ＰＲ文です。

🎮 追加機能：シャッフルゲームとフリップカードゲーム

このPRでは、ユーザー体験を向上させる2つの新しいインタラクティブなミニゲームを追加しました。

🎨 デザインとUXの改善

*   ダークモード対応のカードデザイン: 
    *   `#divFlipCard` に対して、画面全体のレイアウト (`body` の flex 設定) と競合する問題を解消。
    *   `!important` を使用した明確なスタイル定義により、背景色（ダークグレー）、枠線（赤みのあるダーク）、発光効果（box-shadow）を確実に適用。
    *   宇宙をテーマにした視覚効果を実装し、没入感を向上。

*   インタラクティブなボタンエフェクト:
    *   ホバー時とクリック時のアニメーション（スケール変化、背景グラデーション、発光）を追加。
    *   `@media (hover:hover)` を使用し、タッチデバイスとマウスデバイスでの最適な挙動を分けました。

✨ 新機能

1.  シャッフルゲーム (Swap Game)
    *   自然 (Nature)、動物 (Animal)、ストーリー (Story) の3つのカテゴリからランダムな3つの絵文字を生成・表示します。
    *   各カテゴリごとに専用の配列を持ち、Fisher-Yatesシャッフルアルゴリズムを使用して、偏りのない真のランダム性を確保しています。
    *   結果は `→` で繋いで視覚的にわかりやすく表示されます。

2.  フリップカードゲーム (Flip Card Game)
    *   カテゴリ別に整理された質問と回答のペアを、カード形式で表示・確認できます。
    *   カードをクリックすると表/裏（Q/A）を切り替え、ボタンで次のカードへ進めます。
    *   カードをすべて消費すると自動的にシャッフルされ、無限に遊ぶことができます。
    *   カテゴリフィルター（全部、変身、その他）に対応し、ユーザーの好みに合わせて学習内容を絞り込めます。
  
 🌌 没入感のある背景アニメーション

*   動的な星空背景:
    *   CSSの `radial-gradient` を駆使して、色とりどりの星（オレンジ、緑、ピンク、水色、白）をプログラム生成のように配置。
    *   2つのアニメーションを同時実行:
        1.  `moveDiagonal`: 背景画像を対角方向にゆっくり移動させ、宇宙空間を漂うような流動的な動きを演出。
        2.  `twinkle`: 星の透明度（opacity）を0.4と1の間で揺らぎさせ、リアルな星の瞬き（シンリン）効果を実装。
    *   `position: fixed` と `z-index: -1` を使用し、コンテンツの背後で常に動き続ける没入感のある背景を構築。
    *   `will-change` プロパティを活用し、アニメーション時のパフォーマンスを最適化。

🛠️ 技術的なポイント

*   モダンなJavaScript: `const` / `let`、アロー関数、テンプレートリテラル、配列のデストラクチャリングを効果的に使用。
*   アルゴリズム: 両方のゲームでFisher-Yatesシャッフルを実装し、均等な確率でランダムな結果を生成。
*   モジュール化: 画面切り替え (`switchscreen.js`)、シャッフルロジック (`swap.js`)、カード描画 (`flip.js`) をファイルごとに分離し、保守性を向上。
*   レスポンシブ対応**: フレックスボックス (`display: flex`) を活用し、中央配置や高さを動的に調整。

📱 モバイルファーストとPWA対応

*   レスポンシブ設計の徹底:
    *   `<meta name="viewport">` を設定し、あらゆるデバイス（スマホ、タブレット、PC）で最適化された表示を実現。
    *   `device.css` を独立して読み込むことで、デバイス固有の調整を柔軟に管理。

*   PWA（Progressive Web App）対応:
    *   `manifest.json` と `sw.js` (Service Worker) を実装し、Webサイトをインストール可能なアプリとして機能させました。
    *   オフライン環境でも基本的な動作が可能に（キャッシュ戦略による）。
    *   ネイティブアプリのような体験:
        *   `apple-touch-icon` と `theme-color` (`#4B0082`：インディゴ) を設定し、ホーム画面追加時のアイコンやブラウザのアドレスバーの色を統一。
        *   `icon192.png` を使用し、高解像度ディスプレイでの表示も美しく保つ。

*   **モジュール化されたスタイル**:
    *   `body.css` (基本構造), `stars.css` (背景アニメーション), `device.css` (デバイス調整) に役割を分けることで、保守性と可読性を大幅に向上。

📂 変更ファイル

*   `index.html`: 2つの新しいゲーム画面とボタンの追加。
*   `swap.js`: シャッフルゲームのロジック実装。
*   `flip.js`: フリップカードゲームのロジック、データ管理、シャッフル機能。
*   `switchscreen.js`: 画面遷移と状態管理の強化。
*   `style.css`: ゲーム画面のスタイル調整（背景、枠線、フォントサイズなど）。
